Publications

Approximate Orthogonal Gradient Descent for Continual Reinforcement Learning
Sanghun Park, SooJean Han
CKAIA 2025
ABSTRACT: While there has been abundant literature on continual learning for supervised learning scenarios, robotic control scenarios introduce a number of challenges including combination with reinforcement learning and continuous state/action spaces. This paper propoposes Approximate Orthogonal Gradient Descent (AOGD), a novel continual reinforcement learning method which efficiently approximates gradient orthogonalization through cosine similarity-based sampling and weighting, constraining new task gradients to preserve previous task performance while enabling efficient learning of new tasks. We numerically evaluate AOGD on multiple robot environments performing consecutive tasks, and demonstrate that AOGD significantly reduces catastrophic forgetting and achieves more stable learning compared to existing continual learning baselines.
